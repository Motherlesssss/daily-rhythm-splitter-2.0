# 理想汽车 - 商机日节奏拆分工具（可视化版）

## 工具简介

这是一款专为理想汽车获客运营团队设计的商机预测与分配工具。基于历史销售数据，智能预测未来月份每天的商机分配比例，并支持**可视化拖拽调整**，帮助你快速制定精准的日节奏计划。

**核心优势**：
- 📊 **可视化图表**：直观展示每日商机趋势
- 🎯 **拖拽即调**：鼠标拖动即可调整任意一天的目标量
- 🤖 **智能分配**：自动将调整平衡到其他天，确保月度目标不变
- 🔒 **数据安全**：纯浏览器运行，数据不上传任何服务器
- 📈 **实时反馈**：调整后立即看到各车型的分配结果

---

## 功能亮点

### 🎨 汇总可视化

所有车型的每日总量以**柱状图**形式展示，每根柱子代表一天的商机总量：
- 蓝色柱子：未调整的原始预测值
- 橙色柱子：手动调整后的值
- 柱子顶部直接显示数值，无需悬浮

### 🖱️ 拖拽调整

点击柱状图上下拖动，实时调整当天总量：
- 拖动时图表和表格**同步更新**（50ms刷新）
- 系统自动按原始占比分配到各车型
- 其他天自动重新计算，确保月度总量不变

**举例**：
```
调整前：3月15日 450辆
你拖动到：3月15日 600辆（+150辆）

系统自动：
• W01增加：450×40% → 600×40% = 60辆
• W02增加：450×35% → 600×35% = 52辆
• X01增加：450×25% → 600×25% = 38辆
• 其他26天按比例减少150辆

结果：
✓ 各车型月度总量不变
✓ 第15天达到你设定的600辆
✓ 所有数值为整数（无小数）
```

### 📊 多车型管理

支持同时管理多个车型（W01、W02、X01等）：
1. **分车型输入目标**：为每个车型设定月度目标量
2. **切换到"汇总"标签**：查看所有车型的每日总量趋势
3. **拖拽调整总量**：系统自动按比例分配到各车型
4. **导出两种格式**：
   - 标准格式：每个车型一个sheet
   - 横向格式：日期作为列，便于对比

### 🧠 智能算法

基于历史数据的5大智能特性：
- ✅ **去年同期优先**：双11、618等特殊日期的数据比"周几"更准
- ✅ **过滤异常值**：自动去掉最高10%和最低10%的极端数据
- ✅ **近期加权**：最近的数据权重更高，自动适应市场变化
- ✅ **月末冲量**：自动学习你们的月末集中规律
- ✅ **特殊加成**：促销活动×1.5倍，新车发布×1.8倍

## 使用步骤

### 第1步：上传历史数据

准备Excel文件，包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 日期 | YYYY-MM-DD格式 | 2024-10-01 |

| 车型 | 车型编号 | W01、X04 |

**数据量建议**：
- 📅 **推荐12个月**：历史数据越多，预测越准（准确率85-90%）
- ⚠️ **少于3个月**：建议勾选"忽略历史同期数据"

---

### 第2步：选择预测月份

1. 选择**预测年份**和**月份**
2. 系统自动标记工作日、周末、法定节假日

**可选配置**：
- ☑️ **忽略历史同期数据**：春节等特殊月份建议勾选
- ⚙️ **权重调整**：可手动调整工作日/周末/节假日的权重倍数（默认1.0）

---

### 第3步：标记特殊节点（可选）

为特定日期添加标记：

| 类型 | 效果 | 举例 |
|------|------|------|
| 🔄 调休上班 | 按工作日计算 | 国庆假期后的补班日 |
| 🎉 促销活动 | 权重×1.5倍 | 38妇女节促销 |
| 🚗 新车发布 | 权重×1.8倍 | L6上市发布会 |
| 📌 其他 | 权重×1.2倍 | 区域活动 |

---

### 第4步：输入各车型月度目标

点击"生成每日分配比例"后：
1. 切换到**各车型标签**（W01、W02等）
2. 在"月度目标量"输入框输入目标数量
3. 系统自动按预测比例拆分到每一天

**举例**：

1. 切换到**"📊 汇总"标签**
2. 查看每日总量的柱状图趋势
3. **点击并拖动柱子**上下移动，调整当天总量

   - 图表上的数值变化
   - 下方表格的分配结果

**调整提示**：
- 橙色边框 = 已调整的天

- 点击"重置调整"可恢复初始状态

**📊 横向格式**：

## 数据完整性保证

### 月度守恒原则

无论如何调整，系统确保：
- ✅ **每个车型月度总量 = 输入的目标量**
- ✅ **每日汇总 = 所有车型当日之和**
- ✅ **所有数值为整数（无小数）**

### 调整逻辑

**当你拖动某天的总量时**：
1. 该天按各车型原始占比分配
2. 其他未调整天按原始比例重新计算
3. 确保各车型月度总和不变

**精确整数分配算法**：
```
1. 向下取整所有分配值
2. 计算总差额 = 目标值 - 已分配值
3. 按余数降序排列
4. 将差额分配给余数最大的天/车型
5. 结果：总和精确等于目标，全部为整数
```

## 常见问题

### Q1：为什么我调整第15天后，其他天也变了？
**A**：为了保持月度总量不变，增加某天后，其他天会等比例减少。这确保各车型的月度目标精确达成。

### Q2：调整后总量会变吗？
**A**：不会。各车型的月度总量始终等于你输入的目标量，只是每天的分配比例变了。

### Q3：可以调整多少天？
**A**：没有限制。你可以调整任意多天，系统会自动重新计算所有未调整天。

### Q4：如何恢复初始状态？
**A**：点击"重置调整"按钮，清空所有手动调整，恢复到算法预测的初始状态。

### Q5：为什么有些天是整数，有些是小数？
**A**：输入目标量后，所有拆分量都是整数。如果看到小数，可能是在"只显示比例"的模式下。

### Q6：拖拽不精确怎么办？
**A**：按住鼠标慢慢移动，可以精确到个位数。也可以拖到大概位置后，观察柱子顶部显示的数值。

### Q7：导出的Excel包含调整后的数据吗？
**A**：是的。导出时使用的是调整后的最终数据，包含所有手动调整。

### Q8：可以只调整某个车型吗？
**A**：当前版本只支持调整汇总总量，系统自动按原始占比分配到各车型。如需单独调整某车型，建议导出后在Excel中手动修改。

### Q9：为什么柱状图有蓝色和橙色？
**A**：
- 蓝色 = 未调整的原始预测值
- 橙色 = 手动拖拽调整后的值

橙色柱子还有橙色边框，方便快速识别哪些天被调整过。

### Q10：预测准确率有多高？
**A**：
- 历史数据充足（12个月）+ 市场稳定 → 准确率约85-90%
- 历史数据少（3-6个月） → 准确率约70-80%
- 遇到重大变化（新政策、疫情等） → 需要人工判断修正
